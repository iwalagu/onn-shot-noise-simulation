# ONN Shot Noise Simulation

Simulation of a 4×4 MZI mesh optical neural network under Poisson 
shot noise, accompanying the paper "Thermodynamic Limits of Optical 
Neural Networks and the Standard Quantum Limit for AI Energy."

## Dependencies
Julia 1.x with the following packages:
- Plots.jl
- Distributions.jl  
- LinearAlgebra (standard library)

## How to run
julia onn_simulation.jl
